# MP3_player

개발 툴: Keil uVision5

개발 언어: C언어

목적: STM32F407에서 FreeRTOS, uC/GUI, FATFS 라이브러리/ VS1003을 활용한 MP3 플레이어 펌웨어

사용 방법:
1. 터치 LCD에서 파일을 선택하여 재생
2. TFT-LCD UI는 https://github.com/NSJ-student/ImageToArray 및 https://github.com/NSJ-student/MAKE_TFTLCD_GUI 사용
